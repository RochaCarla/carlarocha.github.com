Em 2024, o **LAPPIS (Laboratório Avançado de Produção, Pesquisa e Inovação em Software)/LabLivre** foi selecionado como organização participante do [Google Summer of Code (GSoC)](https://summerofcode.withgoogle.com/archive/2024/organizations/lappis), um programa global do Google que conecta estudantes desenvolvedores a projetos de código aberto.

## Sobre o Google Summer of Code

O Google Summer of Code é um programa anual que oferece bolsas para estudantes trabalharem em projetos de software livre durante o verão. O programa tem como objetivos:

- **Introduzir estudantes ao desenvolvimento de software livre**
- **Conectar estudantes com mentores experientes**
- **Produzir código de qualidade para projetos open source**
- **Fortalecer comunidades de software livre**

## Participação do LAPPIS

A seleção do LAPPIS como organização participante representa um reconhecimento internacional do trabalho desenvolvido pelo laboratório na área de software livre. Como organização mentora, o LAPPIS ofereceu projetos nas seguintes áreas:

- **DevOps e Infraestrutura**
- **Plataformas de Participação Social**
- **Chatbots e Processamento de Linguagem Natural**
- **Engenharia de Dados**

## Impacto

A participação no GSoC 2024 permitiu:

- Mentoria de estudantes internacionais em projetos de código aberto
- Contribuições significativas para os projetos mantidos pelo laboratório
- Fortalecimento da comunidade de software livre brasileira
- Conexão com a rede global de desenvolvedores open source
