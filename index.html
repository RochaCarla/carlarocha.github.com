$ cat research

I am specifically interested in program analysis, parallel libraries, software evolution, source code mining, refactoring tools, user-centered design, and programmer productivity. A common thread in my research is understanding how programmers use parallelism in their code. Based on this understanding, my research goal is to develop techniques and software tools that improve parallel programmer productivity and make parallel programming accessible to all programmers.
